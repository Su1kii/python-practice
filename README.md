# python-practice

Small Python practice projects, exercises, and notes.

## Structure

- `Interviewprep/` - interview-style practice problems and solutions.
- `Payment Receipt Processor/` - scripts and utilities for receipt processing practice.

## How to use

1. Pick a folder.
2. Open a script and run it with Python (3.10+ recommended).
3. Some folders may include their own notes or requirements.

## Notes

- This repo is a learning sandbox; expect ongoing updates and experiments.
